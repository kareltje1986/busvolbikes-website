Toont de actuele fietsenvoorraad van Bus vol Bikes in WordPress.

== Beschrijving ==

Met deze plugin toon je eenvoudig de actuele fietsenvoorraad van Bus vol Bikes op je WordPress website.

Kenmerken:
*   Mooie grid-weergave van fietsen met foto's
*   Filter op gebruikt/nieuw
*   Automatische updates vanuit Fietsenwijk.nl
*   Shortcode ondersteuning
*   Gutenberg blok
*   Widget voor sidebar

== Installatie ==

1. Upload de plugin naar `/wp-content/plugins/busvolbikes-voorraad/`
2. Activeer de plugin via het 'Plugins' menu in WordPress
3. Gebruik de shortcode `[busvolbikes_voorraad]` op een pagina
4. Of voeg het "Bus vol Bikes - Voorraad" blok toe in Gutenberg

== Gebruik ==

**Shortcode:**

`[busvolbikes_voorraad]` - Toont alle fietsen

`[busvolbikes_voorraad limit="6"]` - Toont maximaal 6 fietsen

`[busvolbikes_voorraad filter="gebruikt"]` - Toont alleen gebruikte fietsen

**Gutenberg Blok:**

Voeg het "Bus vol Bikes - Voorraad" blok toe. In de sidebar kun je instellingen aanpassen zoals het aantal fietsen en het standaard filter.

**Widget:**

Ga naar Weergave → Widgets en voeg de "Bus vol Bikes - Voorraad" widget toe aan je sidebar.
